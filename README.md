# CLI-инструмент для выгрузки данных из DNSDumpster в Excel

Документация к консольному скрипту для сбора информации по доменам через API сервиса DNSDumpster. Результаты сохраняются в структурированную Excel-книгу с автофильтрами и автоматическим выравниванием колонок под размер данных.

**Made by cred0core**

---

# Требования и установка

Перед запуском скрипта необходимо установить библиотеки `requests` (для запросов к API) и `openpyxl` (для генерации Excel-файлов):

```bash
pip install requests openpyxl
```

Также для работы потребуется валидный API-ключ от DNSDumpster.

---

# Варианты запуска

## Вариант 1. Передача доменов списком в строке команды

```bash
python callDumpster.py -k ВАШ_API_КЛЮЧ -d example.com example.org example.net
```

## Вариант 2. Чтение доменов из текстового файла

Создайте текстовый файл (например, `domains.txt`), где каждый домен указан с новой строки, и укажите путь к нему:

```bash
python callDumpster.py -k ВАШ_API_КЛЮЧ -f domains.txt
```

---

# Аргументы командной строки

Посмотреть встроенную справку по всем ключам можно командой:

```bash
python callDumpster --help
```

## Основные параметры

| Флаг | Полное имя | Описание |
|------|------------|----------|
| `-k` | `--api-key` | API-ключ DNSDumpster. Можно не вводить вручную, если задана переменная окружения. |
| `-d` | `--domains` | Список доменов для анализа через пробел. Взаимоисключающий с `-f`. |
| `-f` | `--file` | Путь к файлу со списком доменов. Взаимоисключающий с `-d`. |
| `-o` | `--output` | Путь и имя выходного Excel-файла. По умолчанию: `dnsdumpster_report.xlsx`. |
| `-p` | `--proxy` | Строка прокси для работы через корпоративный прокси-сервер. |

---

# Работа в корпоративной среде через прокси

Если в вашей инфраструктуре доступ к внешним ресурсам осуществляется через прокси-сервер, передайте его параметры через ключ `-p`:

```bash
python callDumpster.py -k ВАШ_API_КЛЮЧ -d example.com -p "http://username:password@proxy_address:port"
```

---

# Переменные окружения

Чтобы не передавать API-ключ в командной строке и не сохранять его в истории терминала, можно использовать переменную окружения `DNSDUMPSTER_API_KEY`.

## Linux / macOS

```bash
export DNSDUMPSTER_API_KEY="ваш_api_ключ"
python callDumpster.py -d example.com
```

## Windows (PowerShell)

```powershell
$env:DNSDUMPSTER_API_KEY="ваш_api_ключ"
python callDumpster.py -d example.com
```

# Структура выходного Excel-файла

После завершения работы создаётся Excel-книга с тремя вкладками, оформленными в едином стиле:

- Темно-синие заголовки
- Автофильтры
- Закрепленная первая строка
- Автоматическая подгонка ширины столбцов

## Summary

Сводная статистика по каждому анализируемому домену:

- Количество A-записей
- Количество NS-записей
- Количество MX-записей
- Количество CNAME-записей
- Количество TXT-записей

## Hosts

Подробная информация по найденным хостам:

- FQDN
- IPv4-адрес
- Страна
- ASN
- PTR-запись
- HTTP Server Banner
- HTTPS Server Banner
- Заголовок страницы (Title)
- Applications (определённое ПО)

## TXT

Сырые TXT-записи, сгруппированные по доменам, для анализа:

- SPF
- DMARC
- DKIM
- Токены подтверждения владения доменом
- Другие TXT-записи
